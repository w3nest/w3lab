## Installation, Build, Test

To install the required dependencies:

```shell
yarn
```

---

To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```

---

To run tests:

```shell
yarn test
```

To measure test coverage:

```shell
yarn test-coverage
```

---

To start the development server:

```shell
yarn start
```

---

To generate code's documentation:

```shell
yarn doc
```
